# 📱 TimeMaster — мобільний додаток для залікової роботи

TimeMaster — це мобільний додаток для керування часом на React Native (Expo), створений у межах залікової роботи з дисципліни "Розробка інтерфейсів користувача". Додаток містить секундомір із фіксацією кіл, таймер зворотного відліку зі звуковим та вібраційним сповіщенням, а також гнучкі налаштування теми, звуку і вібрації.

## ✨ Основний функціонал

- **Секундомір**: запуск, пауза, скидання та фіксація кіл із точністю до мілісекунд; підсвічування найшвидшого та найповільнішого кола
- **Таймер зворотного відліку**: колесо вибору годин/хвилин/секунд, швидкі пресети (1 хв, 3 хв, 5 хв, …, 1 год), круговий індикатор прогресу та автоматичний попап зі звуком після завершення відліку
- **Налаштування теми**: ручне перемикання між світлою та темною темами або автоматично за системою
- **Власні звуки**: дефолтний сигнал + до 5 звуків, імпортованих із пристрою
- **Вібрація**: вимкнено або серія імпульсів при спрацюванні таймера
- **Екран "Про додаток"**: інформація про проєкт

## 🛠 Стек технологій

- **React Native** — основний фреймворк для мобільної розробки
- **Expo (SDK 54)** — швидкий старт, збірка та тестування
- **React Navigation (Bottom Tabs)** — навігація між трьома основними екранами
- **expo-audio** — відтворення звукових сповіщень
- **expo-document-picker / expo-file-system** — імпорт власних аудіофайлів із пристрою
- **expo-haptics** — тактильний зворотний зв'язок
- **AsyncStorage** — збереження налаштувань між сесіями
- **react-native-svg** — кругова дуга прогресу на екрані таймера
- **IBM Plex Mono** — шрифт для відображення часу

## 📂 Структура проєкту

```
TimeMaster/
├── App.tsx
├── README.md
├── package.json
└── src/
    ├── constants/    — кольори, шрифти, ключі AsyncStorage, звукові теми
    ├── types/        — TypeScript інтерфейси (LapEntry, ThemeMode, VibrationMode, CustomSound)
    ├── utils/        — форматування часу, аудіо-хелпер
    ├── hooks/        — useStopwatch, useCountdown, useTheme, useSettings
    ├── components/   — TickDial, CustomButton, ScreenHeader, LapItem
    ├── screens/      — StopwatchScreen, TimerScreen, SettingsScreen
    └── navigation/   — RootNavigator (Bottom Tabs)
```

Пояснення структури:

- `App.tsx` — точка входу, завантаження шрифтів, провайдери теми та налаштувань
- `src/screens/` — екрани застосунку (секундомір, таймер, налаштування)
- `src/hooks/` — вся бізнес-логіка (відлік часу, теми, налаштування)
- `src/components/` — спільні UI-компоненти
- `src/utils/` — допоміжні функції (форматування, аудіо)

## 📖 Як користуватися (User Guide)

**Секундомір**
1. Натисніть велику кнопку ▶ для запуску.
2. Під час роботи кнопка «Коло» фіксує проміжний час — список кіл оновлюється в реальному часі.
3. Кнопка «Скидання» обнуляє все після зупинки.

**Таймер**
1. Встановіть час колесами або оберіть швидкий пресет.
2. Натисніть ▶ — розпочнеться зворотний відлік із круговим індикатором.
3. Після закінчення часу пролунає звуковий сигнал (та вібрація, якщо увімкнено) і з'явиться попап «Час стік» з кнопкою OK.

**Налаштування**
- *Зовнішній вигляд*: оберіть світлу, темну тему або перемкніть «Автоматично за системою».
- *Звукове сповіщення*: обирайте дефолтний звук або додайте до 5 власних MP3 із пристрою; непотрібні видаляйте кнопкою 🗑️.
- *Вібрація*: «Вимкнено» або «Серія імпульсів».

## 🚀 Встановлення та запуск локально

Клонувати репозиторій:

```bash
git clone https://github.com/Volodymyr-Zhuchenko/TimerMaster.git
```

Перейти в директорію проєкту:

```bash
cd TimerMaster
```

Встановити залежності:

```bash
npm install
```

Запустити проєкт:

```bash
npx expo start
```

Тестування:
- встановіть додаток **Expo Go** на телефон
- відскануйте QR-код, який з'явиться в терміналі
- або запустіть проєкт на емуляторі Android / симуляторі iOS
