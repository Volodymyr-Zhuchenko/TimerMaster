# 📝 TimeMaster — мобільний додаток для залікової роботи

TimeMaster — це мобільний додаток-таймер із секундоміром, створений на React Native (Expo) у межах залікової роботи з дисципліни "Розробка інтерфейсів користувача". Додаток підтримує точний відлік часу, фіксацію кіл, настроювані таймери зворотного відліку зі звуковими сповіщеннями та вибір теми інтерфейсу.

## ✨ Основний функціонал

- **Секундомір**: запуск, пауза, скидання та відстеження кіл (Laps) з точністю до мілісекунд.
- **Таймер зворотного відліку**: зручний вибір часу, круговий індикатор прогресу, пауза, скидання та автоматичне звукове сповіщення після завершення відліку.
- **Налаштування**: перемикання між світлою та темною темами оформлення, а також вибір звукових тем для сповіщень.

## 🛠 Стек технологій

- **React Native** — основний фреймворк для мобільної розробки
- **Expo** — швидкий старт, збірка та тестування
- **React Navigation (Bottom Tabs)** — навігація між трьома основними екранами
- **expo-av** — робота з аудіо для відтворення звукового сигналу
- **AsyncStorage** — збереження стану додатку між сесіями
- **react-native-svg** — кругова дуга прогресу на екрані таймера
- **IBM Plex Mono** — шрифт для відображення часу

## 🚀 Запуск

```bash
npm install
npx expo start
```

Відскануйте QR-код у застосунку **Expo Go** на телефоні, або запустіть на емуляторі Android/iOS.

## 📁 Структура проєкту

```
src/
├── constants/   — кольори, шрифти, ключі AsyncStorage, звукові теми
├── types/       — TypeScript інтерфейси (LapEntry, ThemeMode, SoundTheme)
├── utils/       — форматування часу, аудіо-хелпер
├── hooks/       — useStopwatch, useCountdown, useTheme, useSettings
├── components/  — CircularProgress, CustomButton, TimeDisplay, LapItem
├── screens/     — StopwatchScreen, TimerScreen, SettingsScreen
└── navigation/  — RootNavigator (Bottom Tabs)
```
